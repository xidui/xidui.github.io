mysql mongodb性能测试

2015-09-27 Tech matlab, mongodb, mysql, python Comments Word Count: 26k(words) Read Count: 1:36(minutes)

对mysql和mongodb做一个简单的调研。

测试环境

数据库版本

mongodb: 3.0.5 无任何优化，单机测试
mysql: 5.6.26 无任何优化，单机测试

环境

CPU: 2.2 GHz Intel Core i7
内存: 16G DDR3
操作系统: OS X Yosemite 10.10.5
python: 3.5.0

数据库接口驱动

mongodb: PyMongo 3.0.3
mysql: PyMySQL 0.6.6

测试指标

  • 插入
  • 排序
  • 搜索

测试脚本

测试思路如下：

  • 清空数据表
  • 插入100次数据，每次10000条
  • 统计每次插入的时间
  • 每次插入后，取出排序后最大的10条记录，并计时
  • 每次插入后，随机搜索10000条，并计时，统计QPS
